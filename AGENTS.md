# AGENTS ルール

## 基本方針
- 出力のベース言語は日本語とする。
- ユーザ向けの説明、確認、質問、進捗報告、要約は日本語で行う。
- ファイルに出力する文書、見出し、説明、注記などのユーザ向け文面も日本語で記載する。

## 日本語化の対象
- ユーザとのやり取り全般
- 進捗報告やレビューコメント
- ドキュメント、テンプレート、レポートなどの成果物
- ファイルへ出力する要約、説明、見出し、ラベル

## 原文を維持するもの
- ソースコード、識別子、関数名、クラス名、変数名
- コマンド、CLI 出力、ログ、エラーメッセージ
- API 名、設定キー、環境変数名、URL、パスなど固定の文字列

## 補足
- 正確性や互換性を損なう要素は原文を維持し、必要に応じて日本語で補足する。
- ユーザから別の言語指定があった場合は、その最新の指示を優先する。
