# 出力例

## 例 1
```text
fix__: ログイン失敗時のエラーメッセージ表示を修正

- 概要: 認証失敗時にエラートーストを表示
- Why: 失敗理由が分からず再試行しづらい状態の解消
```

## 例 2
```text
docs_: READMEにローカル開発手順を追記

- 概要:
  - 初回セットアップ手順を追記
  - 起動確認手順を追記
- Why:
  - 新規参加者が環境構築で詰まりやすい状況
  - 参照先分散の解消
```

## 例 3
```text
feat_: スマホ長押しで動画カード操作メニューを追加

- 概要:
  - タッチ長押しで動画カードの操作ポップアップを表示
  - 長押し後の誤タップ抑止と関連テストを追加
- Why:
  - スマホでは右クリック導線が存在しないため
  - モバイルでも PC と同じ操作を使えるようにするため
```

## 完成形のサンプル
```text
feat_: ユーザー招待メール再送機能を追加

- 概要:
  - 管理画面に再送操作を追加
  - 招待メール再送 API を追加
- Why:
  - 期限切れや未達時の再招待手段が必要
  - 手動対応負荷の軽減
```
